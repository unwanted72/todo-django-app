# Django Todo App

This is a simple Todo Application built using Django.

## Features

- User Login / Logout
- Add Task
- Update Task
- Delete Task
- Task Status (Completed / Pending)
- Due Date for Tasks

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap

## Installation

1. Clone the repository

git clone https://github.com/unwanted72/todo-django-app.git

2. Go to project folder

cd todo-django-app

3. Install dependencies

pip install -r requirements.txt

4. Run server

python manage.py runserver
